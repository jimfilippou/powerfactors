## Getting Started

### Prerequisites

- Node.js
- pnpm or npm or yarn

### Installation

1. Clone the repository
```bash
git clone git@github.com:jimfilippou/powerfactors.git
```

2. Install dependencies
```bash
pnpm install
# or
npm install
# or
yarn install
```

3. Start the development server
```bash
pnpm dev
# or
npm run dev
# or
yarn dev
```

The application will be available at `http://localhost:5173`
